# LabPilot Docs Index

This folder contains planning and execution docs for the LabPilot hackathon MVP.

## Documents

- `prd.md`: product requirements, architecture, demo story, and roadmap.
- `focus.md`: execution priority and milestone order (modeling first).
- `available_tooling.md`: sponsor tool mapping and scope guardrails.
- `dataset.md`: dataset selection guidance and constraints.
- `modeling_quickstart.md`: commands to run the modeling loop locally.
- `llm_reasoning_approach.md`: current LLM status and planned reasoning-layer integration.
- `progress_so_far.md`: full implementation log, current results, and next steps.
- `frontend_build_guide.md`: frontend/API integration and navigation spec.

## Suggested reading order

1. `focus.md`
2. `prd.md`
3. `available_tooling.md`
4. `dataset.md`
